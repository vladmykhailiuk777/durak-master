# DurakMaster — «Дурень»

> Консольна гра «Дурень» на C++

---

## Про проєкт

**DurakMaster** — це термінальна гра для 2–6 гравців, написана на C++. Підтримує спільну гру кількох людей за одною клавіатурою. Реалізовано класичний режим **Підкидного Дурня** з повноцінною ігровою логікою: козирі, підкидання, відбивання та перевірка переможців.

---

## Як грати

1. Запустіть гру та оберіть **«Нова гра»** з головного меню
2. Введіть імена гравців (або залиште порожнім для імені за замовчуванням)
3. Гравці по черзі атакують та захищаються, вводячи номер карти

### Керування

| Дія | Введення |
|-----|----------|
| Вибрати карту | `1`–`N` (номер карти в руці) |
| Завершити атаку / Взяти карти | `0` |

---

## Правила гри (Підкидний Дурень)

1. Кожному гравцю роздається по **6 карт**
2. Мета — першим **позбутися всіх карт**
3. Атакуючий ходить будь-якою картою; захисник відбиває старшою картою тієї ж масті або козирем
4. Сусіди можуть **підкидати** карти того ж рангу (максимум 6 карт на столі)
5. Якщо захисник відбився — карти йдуть у **відбій**, він стає новим атакуючим
6. Якщо не відбився — **забирає всі карти** зі столу, хід переходить далі
7. Після кожного раунду гравці добирають карти з колоди до 6
8. Хто останній залишився з картами — **Дурень** 

---

### Основні класи

| Клас | Відповідальність |
|------|-----------------|
| `Card` | Карта (масть, ранг, козир, порівняння) |
| `Deck` | Колода: генерація, перемішування, роздача |
| `Player` / `HumanPlayer` | Гравець та його дії |
| `Table` | Ігровий стіл (атакуючі та захисні карти) |
| `GameMode` / `ThrowingDurak` | Стратегія ігрового режиму |
| `GameSession` | Оркестрація гри: ходи, роздача, перевірка перемоги |
| `GameResult` | Результат партії |

Код використовує **поліморфізм** (`GameMode` як абстрактний базовий клас) для легкого розширення на нові режими гри.
